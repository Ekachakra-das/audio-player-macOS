# Audio Player for macOS

Легковесный аудиопроигрыватель для macOS, разработанный специально для удобного прослушивания аудиокниг и лекций с автоматическим сохранением позиции воспроизведения.

## Установка и подготовка

Для работы приложения необходим **Python 3** и библиотека **pywebview**.

### 1. Установка Python
Если у вас еще не установлен Python 3, скачайте его с [python.org](https://www.python.org/downloads/) или установите через Homebrew:
```bash
brew install python
```

### 2. Установка зависимостей
Откройте Терминал и выполните:
```bash
pip3 install pywebview
```

## Как использовать

Приложение максимально просто в использовании:
1. **Портативность**: Просто запишите (скопируйте) `Srimad Bhagavatam.app` (или переименованную копию) в любую папку с музыкой либо с аудиокнигами, и оно сразу будет там работать.
2. **Запуск**: Запустите приложение. Оно автоматически отсканирует текущую директорию и все ее подпапки, после чего выведет список треков.
3. **Смена папки вручную**: Вы можете в любой момент сменить рабочую папку прямо из интерфейса. Для этого нажмите кнопку с иконкой папки и плюсом (📁+) справа от заголовка «Folders» в боковом меню.

## Основные возможности

* **Автоматическое сканирование**: При запуске сканирует папку на наличие аудиофайлов (`.mp3`, `.m4a`, `.wav`, `.ogg`, `.flac`).
* **Поддержка структуры папок**: Отображает дерево папок для удобной навигации.
* **Умное сохранение позиции (Resume)**: Автоматически запоминает трек и время.
* **Полная независимость**: Каждая копия приложения хранит свои настройки внутри своего бандла (папка `Contents/Settings`), что позволяет использовать разные копии плеера для разных аудиокниг.
* **Быстрое определение времени**: Длительность треков вычисляется «на лету» в браузере с использованием `IntersectionObserver`.
* **Управление с клавиатуры**: Пробел — пауза/старт, стрелки — перемотка ±10 секунд.

## Структура файлов приложения

```text
AudioPlayer.app/
└── Contents/
    ├── MacOS/
    │   └── launcher          # Точка запуска приложения
    ├── Settings/             # Локальные настройки конкретной копии плеера
    └── Resources/
        ├── app.py            # Серверная логика на Python
        ├── index.html        # Интерфейс плеера
        └── README.md         # Эта инструкция
```
